* Display Possibilities
    * Writing into an HTML element, using innerHTML or innerText.
    * Writing into the HTML output using document.write().
    * Writing into an alert box, using window.alert().
    * Writing into the browser console, using console.log().

* window.alert() function 
    displays a message box with a specified message and an OK button. 
    It halts the further execution of the code until the user presses the OK button.

* console.log() function 
    is used for printing some output to the console. 
    It does not block the execution of further code and does not require user interaction.

* window object, alert() method 
    * window object is the global scope object | https://www.w3schools.com/jsref/obj_window.asp
    * The alert() method displays an alert box with a message and an OK button.

* console object, log() method
    * console object provides access to the browser's debugging console | https://www.w3schools.com/jsref/obj_console.asp
    * log()	Outputs a message to the console

* setInterval | https://www.w3schools.com/jsref/met_win_setinterval.asp
    * repeatedly runs a piece of code at a specified time interval (in milliseconds).

    Syntax  setInterval(function, milliseconds)

*   () => { ... } (Arrow Function)
    This is an arrow function — a shorter way to write a function in JavaScript.
    Whatever is inside { ... } will be executed each time the interval is triggered.

    setInterval(() => {
        alert("Please, use me...");
    }, 30000);
