* Document Object Model (DOM) Events | https://www.w3schools.com/jsref/dom_obj_event.asp
    allow JavaScript to add event listener 
    or event handlers to HTML elements.

    Example:
        <button onclick="myFunction()">Click me</button>

        <element event="some JavaScript">

    * In HTML onclick is the event listener | https://www.w3schools.com/jsref/event_onclick.asp
    * myFunction is the event handler       | https://www.w3schools.com/js/js_events.asp


* HTML DOM -- Style | https://www.w3schools.com/jsref/prop_style_backgroundcolor.asp
    Example:
        document.style.backgroundColor = "color"

    * document → is a special object in JavaScript that represents 
                the entire HTML page currently loaded in the browser.

    * document.body → refers specifically to the <body> element of that page.

    * .style.backgroundColor → changes the CSS style of that <body> element 

    * `rgb(${red}, ${green}, ${blue})`
        Backticks ( ) → allow us to use template literals, which are special strings in JavaScript.
    * {variable} → string interpolation

    * RGB values | https://www.w3schools.com/html/html_colors_rgb.asp
